# Loja online - Fazendo arte

## Objetivo

Prova de conceito de uma loja online de produtos para a casa, com cadastro e alteração de produtos, sistema de login e cadastro de usuários.

## Tecnologias utitlizadas
- ReactJs
- Axios
- React-router
- CSS3
